# Shadows

Add or remove shadows from elements with `box-shadow` utilities.

## Basic Usage

- `.shadow-sm` - Adds a small box shadow.
- `.shadow` - Adds a default box shadow.
- `.shadow-lg` - Adds a large box shadow.
- `.shadow-none` - Removes any box shadow.

### Example

<div class="d-flex" style="gap: 2rem; padding: 1rem;">
  <div class="shadow-sm" style="padding: 1rem; background-color: white;">.shadow-sm</div>
  <div class="shadow" style="padding: 1rem; background-color: white;">.shadow</div>
  <div class="shadow-lg" style="padding: 1rem; background-color: white;">.shadow-lg</div>
</div>

```html
<div class="shadow-sm">...</div>
<div class="shadow">...</div>
<div class="shadow-lg">...</div>
```
